# WeatherForecast
Как запустить:
* склонировать к себе git clone https://github.com/OlegTar/WeatherForecast.git
* перейти в папку PowerTestTask/PowerTestTask.client в консоли.
* запустить npm install (если такая команада не находится, то скорее всего nodejs не установлен. Установить отсюда: https://nodejs.org/en у меня версия 22.15.0)
* открыть PowerTestTask/PowerTestTask.sln в Visual Studio
* отметить проект PowerTestTask.Server как запускаемый по умолчанию, правой кнопкой мыши - Set as Startup Project
* запустить проект нажав, зеленую стрелку или F5
* запустится и бэк, и фронт, открыть фронт https://localhost:52799/

### Если не получилось запустить, напишите в Телеграм t.me/OlegTar !
